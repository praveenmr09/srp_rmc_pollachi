DOCUMENTS OF FLEETS WITH EXPIRY ALERT

It is important to keep the documents of all the vehicles safely. The module helps to store the documents of the vehicle such as Vehicle Registration Certificate, Insurance Document, Road Tax Document, Inspection certificate, Fuel Expense receipts etc. And some documents will be expired and it is very important to renew it before it is getting expired. The module helps to view the documents which are going to be expired within 30 days And it will be helpful , if we can download any fleet document, whenever it needs. The module helps it too

Version 16.0 Enterprise & Community

Overview

Upload and add multiple documents of a fleet

Key Highlights!

  • No need any configuration
  • Upload any number of documents
  • Can provide brief description of the document
  • Can provide the expiry date of the document
  • Can set if the document expiry to be notified
  • A report will provide the documents which are going to be expired within 30 days
  • Possible to exclude any document from the expiry notification
  • Download any document, whenever it requires

Features

01

While opening a fleet, the focus will be on the document tab. Click on 'Add Line'



02

Upload document and provide the details such as the name of the document, expiry date etc.



03

Add multiple documents



04

Click on the Reporting - > Documents Expiry Reminder , to view the documents which will get expired within 30 days



05

The list of the documents which are going to be expired and by clicking on it, the details can be viewed. Once it is renewed, the new expiry date can be given from there itself



06

It is possbile to download the documents, whenenver we need, by just clicking on download button
